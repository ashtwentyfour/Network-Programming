to run:

- run node weather_server.js on the terminal
- open http://localhost:3000/Weather.html
- select 'Get Weather Data' after entering a zip code

Note: if the data for the zip code has been cached, the program will return the cached version of the data

- select 'Get Weather Data Cache' to view all the weather data in the cache in JSON format
- select 'Delete Zip Code Data' to delete cache data for a given zip code from the cache
- the 'setTimeout' function will delete expired cache data every 30 minutes
